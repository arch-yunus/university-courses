# 📂 03 — Seçmeli & İleri Uygulama — Hukuk

## 📖 Açıklama
Seçmeli hukuk dersleri, ileri düzey vaka çalışmaları, moot court (mahkeme simülasyonu) ve uzmanlık alanı projeleri.

---

## 📝 Bu Klasöre Nasıl Katkı Yapılır?
1. `DERS_SABLONU.md` (üst dizindeki) dosyasını bu klasöre kopyala
2. Dosyayı `DERS_ADI.md` olarak yeniden adlandır
3. İçini ders notları, ödevler ve kaynaklarla doldur
4. Pull Request aç veya doğrudan push et

## 🗂️ Örnek İçerikler
- KVKK / GDPR Hukuku notları
- Uluslararası Tahkim vaka analizi
- LegalTech ve Yapay Zekâ Hukuku araştırması
- İş Hukuku seçmeli ders notları
