# 📂 02 — Alan Dersleri — Hukuk

## 📖 Açıklama
Hukuk bölümünün özgül ve zorunlu alan derslerine ait notlar: Medeni Hukuk, Borçlar, Ceza, İdare, Ticaret vb.

---

## 📝 Bu Klasöre Nasıl Katkı Yapılır?
1. `DERS_SABLONU.md` (üst dizindeki) dosyasını bu klasöre kopyala
2. Dosyayı `DERS_ADI.md` olarak yeniden adlandır
3. İçini ders notları, ödevler ve kaynaklarla doldur
4. Pull Request aç veya doğrudan push et

## 🗂️ Örnek İçerikler
- Medeni Hukuk I-II notları
- Borçlar Hukuku Genel Hükümler
- Anayasa Hukuku içtihat özeti
- Ceza Hukuku vaka analizleri
